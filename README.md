# SASCA

## Development

```bash
cd frontend
npm run dev
```

```bash
cd backend
npm run dev
```

## Service Mode (SERVICE_MODE)
```bash
ALL -> Menghidupkan semua service pada container
JOB -> Hanya menghidupkan service cronjob atau queue
FE  -> Hanya menghidupkan service frontend
BE  -> Hanya menhidupkan service backend
```
